# **📝 Todo List (Mini Project)**

## **📌 프로젝트 소개**

React 학습을 종합하여 구현한 개인 프로젝트입니다.

단순한 CRUD를 넘어 **시간 표시, 랜덤 명언, Custom Hook, json-server** 등을 적용하며

프론트엔드 기본기를 강화하는 것을 목표로 했습니다.
